This is the first post on **wc notes**.

The goal of this site is simple:

- keep writing in plain Markdown;
- publish fast static pages;
- own the source files and history;
- use free developer-friendly services where possible;
- keep the system easy to migrate.

The initial stack is Astro, GitHub, static hosting, Pagefind search, RSS, sitemap, and GitHub-based comments through Utterances.
