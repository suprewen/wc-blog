A practical free stack for a personal blog:

- **Source control**: GitHub.
- **Static hosting**: Cloudflare Pages, Netlify, or Kinsta Static Site Hosting.
- **DNS**: Cloudflare DNS, deSEC, or Hetzner DNS.
- **Search**: Pagefind for static local search; Algolia if hosted search is needed.
- **Comments**: Utterances, powered by GitHub Issues.
- **Analytics**: Cloudflare Web Analytics or counter.dev.
- **Forms**: Formspree, Formcarry, or Formspark.
- **Newsletter**: Buttondown for a small list.

The default choice for this site is: GitHub + Astro + Cloudflare Pages + Pagefind + Utterances.
